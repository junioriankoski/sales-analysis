# Sales Data Analysis

A Python project that analyzes sales data from a CSV file and generates visualizations.

## Features
- Loads and processes sales data from CSV
- Calculates total revenue per product
- Identifies best-selling product
- Shows revenue breakdown by category
- Generates and saves a bar chart

## Technologies
- Python
- Pandas
- Matplotlib

## How to run
1. Clone the repository
2. Install dependencies: `pip install pandas matplotlib`
3. Run: `python project.py`

## Output example
- Best selling product with total revenue
- Total sales summary
- Revenue by category
- Bar chart saved as `sales_infograph.png`
